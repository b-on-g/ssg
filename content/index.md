# Welcome to $mol SSG

This page was **statically generated** from a $mol application using `$mol_browser`.

## How it works

1. You write content in Markdown files
2. `$mol_text` renders them as rich components
3. `$mol_browser` pre-renders each page to static HTML via Puppeteer
4. Output: static HTML that works without JavaScript

## Features

- Same codebase produces both **static** and **interactive** versions
- All `$mol` components available for custom pages
- Markdown rendered by `$mol_text` with full formatting support
- Zero external dependencies (no Eleventy, no Next.js)
