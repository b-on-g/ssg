# Welcome to Eleventy SSG

This page was **statically generated** using Eleventy.

## How it works

1. You write content in Markdown files with frontmatter
2. Eleventy processes them through a template engine
3. Output: static HTML files

## Features

- Simple Markdown to HTML conversion
- Nunjucks/Liquid templates for layouts
- Data cascade from JSON/YAML
- Zero client-side JavaScript
