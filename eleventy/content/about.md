# About Eleventy SSG

This is the same content as the $mol SSG version, but rendered by Eleventy.

## Architecture

- **Eleventy** reads Markdown + frontmatter
- **Nunjucks** template wraps it in a layout
- **Output**: static HTML, no JavaScript

## Comparison

Eleventy generates static HTML only. For interactivity you need a separate framework (React, Vue, Svelte).

The $mol SSG version produces both static HTML AND an interactive SPA from the same codebase.
